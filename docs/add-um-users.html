Add U-M Users

Bulk adding U-M users to course sections with Canvas Course Manager is an administrator feature that offers some benefits over the People tool method:

  • You can add users from a CSV file.
  • You can create a section on the fly and add users to it.
  • You can add users to multiple preexisting sections at one time.
  • The process does not send email notifications to new course members like the built-in Canvas People tool.

Who Can Add U-M users

Bulk adding U-M users to sections is limited to Canvas administrator roles:

  • Root Account Administrator
  • Sub-Account Administrator
  • Support Consultant
  • Teachers and other course managers can request assistance by contacting 4help@umich.edu.

Tip: Instructors and other roles with People tool access can add U-M users in bulk by entering a list of email addresses or Uniqnames (login ID) in the People tool. Note that users added this way may receive a new course invitation message from Canvas.

How to Add U-M Users to a Single Section

This feature allows administrators to select an existing section or create a new section to accept the users.

  1. Click the Add U-M users button on landing page.
  2. Choose the “1 Section at a Time” option and click Select.
  3. Download the example CSV file, if needed.
  4. Prepare and save your CSV file following the onscreen instructions for header and row content.
  5. Drag and drop, or click the link to upload your saved CSV file.
  6. CSV file data is presented back to you. Verify the results and click Submit.

How to Add U-M Users to Multiple Sections

This feature allows administrators to add U-M users to more than one existing section at the same time. Users are not required to have the same role. New sections cannot be created with this tool.

To use this feature you must include the Canvas section ID (not the section SIS ID) in your CSV file. A list of course section IDs is included with this tool to help with CSV file preparation.

  1. Click the Add U-M users button on landing page.
  2. Choose the “Using Multiple Sections” option, and click Select.
  3. Download the example CSV file, if needed.
  4. Prepare and save your CSV file following the onscreen instructions for header and row content. A list of course section IDs is presented on screen and a CSV version is available for download.
  5. Drag and drop, or click the link to upload your saved CSV file.
  6. CSV file data is presented for you to review. Verify the results, and click Submit.
